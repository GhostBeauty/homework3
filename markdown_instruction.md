## Списки

Чтобы добавить ненумированные списки, необходимо пункты выделить здвездочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Первый
2. Второй
3. Третий
